# Simoné Govender Cosmic Portfolio Game

An interactive cosmic portfolio game experience for Simoné Govender.

The site turns live GitHub Pages projects into destinations in a 3D galaxy map. Each destination opens the original project sandbox without changing its purpose, content, live URL, or repository link.

## Highlights

- Mission Control launch screen
- 3D galaxy map with project planets, stations, moons, nebulae, and satellites
- Holographic project preview panel
- Preserved live project iframe sandboxes
- Quick destination menu for recruiters who prefer direct navigation
- Captain's log, skill constellation, SAP credential section, CV download, and contact signal area
- Local Three.js bundle for reliable GitHub Pages hosting
